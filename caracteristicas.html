Detalles técnicos

Qué incluye miOS

Resumen de componentes, requisitos y opciones de personalización.

Requisitos y rendimiento

Requiere mínimo 2GB de RAM para una experiencia fluida. En pruebas el sistema en reposo suele usar alrededor de 650–690 MB; con más RAM y aplicaciones abiertas el consumo puede subir a 1.2–2.2 GB.

Software incluido

Conjunto de aplicaciones ligeras para tareas diarias: gestor de archivos, navegador, editor de texto, cliente de correo y reproductor multimedia. El objetivo es ofrecer una experiencia usable sin sobrecargar el hardware.

Sistema base y actualizaciones

Basada en Debian estable. miOS no mantiene repositorios APT oficiales; la imagen ISO incluye paquetes seleccionados tomados de Debian y de fuentes comunitarias. Las actualizaciones de seguridad provienen principalmente de los repositorios de Debian; el proyecto publica imágenes y parches mantenidos por la comunidad.

Gestión de paquetes: APT (dpkg/apt). Snaps no incluidos por defecto.

Privacidad y personalización

Sin telemetría activa. La distribución incluye herramientas y temas para personalizar el escritorio XFCE de forma sencilla.

¿Quieres ayudar?

Únete a la comunidad para reportar errores, colaborar en traducciones, creación de imágenes y documentación. Encontrarás soporte en el canal y el chat.
